Future Wireless Communications Program

A comprehensive, three-module program covering Embedded Systems, 5G Networks, and AI/ML applications

01

Digital Design & Embedded Programming

Duration: 2 Months | Fee: ₹45,000

02

Design of 5G Networks with Hands-on

Duration: 4 Months | Fee: ₹75,000

03

AI/ML for 5G and 6G Wireless Communication (Optional)

Duration: 3 Months | Fee: ₹75,000

Detailed Curriculum

Module 1: Advanced Digital Design and Embedded Programming

Covers digital design, programming (assembly/Embedded C), IoT, and Wired Protocols, using hardware platforms like Arduino, Raspberry Pi, and Pico W.

Topics Covered

  • Digital Design
  • Embedded C Programming
  • Python using AI Programming
  • Wired Protocols
  • Data Structures
  • Unmanned Ground Vehicles
  • Inter-chip communication

Learning Outcomes

  • Introduction to PlatformIO, Porting Digital Logic to Arduino Pico W
  • AVR-GCC, Functions, BIT-FIELDS, Pointers, Threads, Socket programming
  • AI Tools with NumPy and Pandas simulations
  • UART/SPI/I2C/RS-485 configuration in embedded systems
  • Pointers and lists for vector/matrix operations
  • Toy car assembly, control, and communication
  • ESP/FPGA/ARM co-processing
Module 2: Design of 5G Networks with Hands-on
  1. Evolution of Mobile Technologies: From 1G to 5G, Towards 6G, Comparison of Generations
  2. Introduction to 5G, Key Technologies, Use Cases: Role of ITU and 3GPP, 5G Usage Scenarios (eMBB, uRLLC, mMTC), SA vs NSA Architectures
  3. 5G Core: Introduction, Key Functions (AMF, SMF, UPF, AUSF, UDM), Service-Based Architecture (SBA), CUPS, Network Slicing, Interfaces (N1-N8)
  4. 5G RAN L2/L3: RAN Split Architecture (CU/DU/RU), O-RAN overview, 5G Interfaces (NGAP, F1AP, Xn, E1, NG), Layer 2 (SDAP, PDCP, RLC, MAC) and Layer 3 (NAS, RRC) Protocols
  5. Physical Layer L1: Key 5G-NR technologies (Hybrid ARQ, OFDM, AMC, MIMO), Radio Interface, Numerologies, Uplink/Downlink Channel Design
  6. 5G SA / NSA Call Flows: Detailed procedures for 5G NSA and SA Call Flows, Registration, and PDU Session Establishment
Module 3: AI/ML for 5G and 6G Wireless Communication (Optional)
  • Introduction to Machine Learning: Overview of supervised, semi-supervised, unsupervised learning, and detailed exploration of regression models
  • Introduction to Wireless Communication (Python): Single Carrier Systems, OFDM, MIMO, and OTFS (Orthogonal Time Frequency Space) systems
  • AI/ML in Wireless Communications: Massive MIMO receiver design, Modulation classification techniques

Program Structure and Application

The program fee is charged per module and collected before the start of each module.

Students have the option to exit after each module, but the fee is non-refundable after payment.

The selection process is based on an offline written test.
